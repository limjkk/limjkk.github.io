문제
대학생 새내기들의 90%는 자신이 반에서 평균은 넘는다고 생각한다. 당신은 그들에게 슬픈 진실을 알려줘야 한다.

입력
첫째 줄에는 테스트케이스의 개수 C가 주어진다.

둘째 줄부터 각 테스트케이스마다 학생의 수 N(1 ≤ N ≤ 1000, N은 정수)이 첫 수로 주어지고, 이어서 N명의 점수가 주어진다. 점수는 0보다 크거나 같고, 100보다 작거나 같은 정수이다.

출력
각 케이스마다 한 줄씩 평균을 넘는 학생들의 비율을 반올림하여 소수점 셋째자리까지 출력한다.

예제 입력 1
5
5 50 50 70 80 100
7 100 95 90 80 70 60 50
3 70 90 80
3 70 90 81
9 100 99 98 97 96 95 94 93 91
예제 출력 1
40.000%
57.143%
33.333%
66.667%
55.556%


```
#include <stdio.h>
#include <stdlib.h>

int main()
{
    int num3;
    scanf("%d",&num3);
    for (int j=0;j<num3;j++)
    {
        double aver=0,aver2=0;;
        int num,i,num2=0;
        scanf("%d",&num);
        int *data =calloc(num,sizeof(int));
        for(i=0;i<num;i++)
        {
            scanf("%d",&data[i]);
            aver += data[i];
        }
        aver2 = aver/num;
        for(i=0;i<num;i++)
        {
            if(data[i] > aver2)
                num2++;
        }
        printf("%.3f%%\n",((double)num2/num*100));
        free(data);
    }
}

```
